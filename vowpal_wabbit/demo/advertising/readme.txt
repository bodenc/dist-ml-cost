        ------ Criteo Conversion Logs ------

This dataset contains feature values and conversion feedback for millions of clicks on display ads.
It served to benchmark algorithms described in the paper "Modeling Delayed Feedback in Display Advertising", KDD 2014 by O.Chapelle (http://olivier.chapelle.cc/pub/delayedConv.pdf)

===================================================

Instructions

 - Go to http://labs.criteo.com/downloads/2014-conversion-logs-dataset/ 
 - Download the tar file after agreeing to the terms and conditions
 - Untar the file 
 - Move data.txt to this directory
 - Run the naive_baseline.py script
